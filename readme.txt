Simulates Car Traffic

Additonal Program  Details
####################
I used the LinkedListApi.h file provided in class

To run my program, make program, then run the executable
with a .txt file with car data

Car Data files should look like this: 
N F 1

The first element is the direction the car is facing either N, S, E or W

The second element is the direction the car is moving, either F (forward), L (left), or R (right)

The last element is the time the car got to the intersection
